## Notes & caveats

- **Classes 6, 7, 8** use the **new NCF-SE 2023 textbooks** (Mathematics = *Ganita Prakash*, Science = *Curiosity*, Social Science = *Exploring Society: India and Beyond*, English = *Poorvi*), rolled out Class 6 in 2024-25 and Classes 7–8 in 2025-26.
- **Class 8 Social Science** currently has only **Part 1 (7 chapters)** published; a Part 2 is expected to follow. Treat as provisional.
- **Classes 9, 10** use the **rationalised** NCERT editions (post-2023 syllabus trimming). Removed chapters (e.g. Class 10 Science: *Periodic Classification of Elements*, *Sources of Energy*, *Management of Natural Resources*; Class 9 Science: *Diversity in Living Organisms*, *Why Do We Fall Ill*, *Natural Resources*) are **not** included.
- **Class 9 NCF-SE 2023 note:** For the 2026-27 session NCERT began issuing new Class 9 books (Science *Curiosity*, Maths *Ganita Prakash*, integrated *Exploring Society*). This seed intentionally uses the **rationalised editions** for Classes 9–10 so the two board-prep classes stay consistent and match the widely-used current chapter set. Revisit when NCF Class 9/10 books are fully in force.
- **Classes 11, 12** reflect the **rationalised** senior-secondary syllabus: Class 11 Chemistry = 9 chapters, Class 12 Chemistry = 10 chapters, Class 11 Physics = 14 (Physical World removed), Class 12 Physics = 14 (Communication Systems removed), Class 11 Biology = 19, Class 12 Biology = 13, Class 11 Maths = 14, Class 12 Maths = 13.
- **Social Science (Classes 9–10)** is listed as a single subject but its chapters span the four NCERT books (History, Geography, Political Science/Civics, Economics), in that order.
- **English (Classes 9–10)** lists the main reader (Beehive / First Flight) prose + poems; supplementary readers (Moments / Footprints Without Feet) are not separately seeded.
- **Commerce (Classes 11–12):** Accountancy, Business Studies and Economics are included as a bonus. Economics combines both books per class (Statistics for Economics + Indian Economic Development for Class 11; Introductory Micro + Macroeconomics for Class 12).
